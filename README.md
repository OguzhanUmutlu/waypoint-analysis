# waypoint-analysis

A toy project to compute the target orientation of a fixed-wing vehicle based on geofencing constraints and waypoints.

## Setup

1. Clone the repository:
   ```bash
   git clone git@github.com:OguzhanUmutlu/waypoint-analysis.git
   cd waypoint-analysis
   ```
2. Download SFML through vcpkg:
   ```bash
   vcpkg install sfml
   ```
3. Open the project in your favorite C++ IDE (e.g., Visual Studio, CLion) and configure it to use vcpkg toolchain.
4. Build the project.
